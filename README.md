## Descripción del proyecto

Aplicación web sencilla para calcular la media de varias notas introducidas por el usuario.

## Objetivo de la práctica

Aprender el uso básico de Git y GitHub, incluyendo:

- Inicialización de repositorios
- Uso de commits
- Trabajo con ramas
- Resolución de conflictos
- Publicación de proyectos en GitHub

## Funcionalidades

- Introducción de notas numéricas
- Cálculo automático de la media
- Soporte para hasta 4 notas
- Mensajes informativos con el resultado
- Interfaz web sencilla en HTML y CSS
